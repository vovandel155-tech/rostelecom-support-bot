## 1. Руководство пользователя

### 1.1. Запуск приложения
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите Streamlit:
   ```bash
   streamlit run all.py
   ```
3. В браузере откройте адрес http://localhost:8501.

### 1.2. Стартовый экран
- Меню «Чат-бот» (и «Аналитика» для админа).
- Выберите тему кнопкой или введите вопрос вручную.
- Нажмите Enter или кнопку «Начать чат».

### 1.3. Ведение диалога
- Бот отвечает на типовые вопросы.
- При низкой уверенности предлагает уточнения.
- «🔙 Сбросить чат» — возвращает к выбору темы.
- «Связаться с оператором» — эскалация запроса.

### 1.4. Оценка ответа и обучение
- После ответа оцените «👍 Полезно» или «👎 Не полезно».
- При «Не полезно» предложите свой вариант ответа.

## 2. Руководство оператора

### 2.1. Запуск панели оператора (Flask)
1. Запустите сервер:
   ```bash
   python server/app.py
   ```
2. Откройте http://127.0.0.1:5000/operator.

### 2.2. Работа с эскалированными запросами
- Список эскалированных запросов.
- Обработка запросов вне интерфейса.
- Переход к аналитике и администрированию по ссылкам.

---

## 3. Руководство администратора

### 3.1. Вход
1. В меню выберите «Аналитика».
2. Введите пароль администратора.
3. После входа доступен раздел «Аналитика» и кнопка «🚪 Выйти».

### 3.2. CRUD-база знаний (Flask)
1. Откройте http://127.0.0.1:5000/admin.
2. Добавляйте, редактируйте, удаляйте записи.
3. Экспортируйте/импортируйте CSV и JSON.

## 4. Техническая документация по архитектуре

### 4.1. Структура проекта
```
/Чат-Бот
├── all.py             # Streamlit: чат + обучение + аналитика
├── chat.py            # Отдельный модуль чат-бота
├── analytics.py       # Отдельный модуль аналитики
├── nltk_setup.py      # Скрипт для NLTK
├── faq.csv            # CSV базы знаний
├── requirements.txt   # Зависимости
├── data/
│   └── db.sqlite      # SQLite база
├── logs/
│   ├── requests.log
│   ├── missed.log
│   ├── escalations.log
│   ├── feedback.log
│   └── suggestions.log
├── server/
│   ├── import_faq.py
│   └── app.py         # Flask: оператор, аналитика, админ
├── .streamlit/
│    └──secrets.toml

```

### 4.2. Компоненты
- **all.py** — объединённый интерфейс.
- **Flask/app.py** — административная панель.
- **SQLite** — таблица `faq`.
- **Логи** — для аналитики и обучения.

### 4.3. Потоки данных
1. `log_request` → `requests.log`
2. NLP-обработка → ответ или `log_missed`
3. `log_escalation` → `escalations.log`
4. `log_feedback` и `log_suggestion` → `feedback.log`/`suggestions.log`
5. Аналитика читает логи и строит дашборд

### 4.4. Зависимости
- Python 3.7+
- `streamlit`, `flask`, `nltk`, `pandas`
- Запуск:
  - Streamlit: `streamlit run all.py`
  - Flask: `python server/app.py`
```
